# Software Requirement Specification

## 1. Purpose
CodeAtlas is a web-based platform for learning programming and solving DSA problems.

## 2. Users
- Student
- Admin

## 3. Functional Requirements

### Authentication
- Register
- Login
- JWT Security

### Course Module
- Add Course
- Add Topics
- View Content

### DSA Module
- Problem List
- Difficulty Filter
- Code Submission

### Compiler
- Run C, C++, Java, Python, JavaScript

### Admin
- Manage Users
- Manage Content

## 4. Non-Functional
- Secure
- Fast
- Scalable
- Mobile Responsive
